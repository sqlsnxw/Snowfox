# [Android Components](../../README.md) > Samples > Acorn Components

A simple application demonstrating the [Acorn Design System](https://acorn.firefox.com/) components for Android.

This app showcases the various components available in the `compose-base`, `ui-icons` and `ui-colors` modules.
