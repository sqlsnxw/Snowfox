Snowfox is a privacy-focused browser built on Firefox.

It ships with a clean dark theme, SearXNG as the default search engine, privacy-focused settings enabled out of the box, and useful extensions already configured.

No telemetry. No sponsored content. No unnecessary extras.

Just a browser.

## Features

* Clean dark interface
* Vertical and horizontal tabs
* SearXNG search by default
* uBlock Origin included
* Browser fingerprinting protections
* Enhanced tracking protection
* Telemetry disabled
* Pocket disabled
* Sponsored content disabled
* Privacy-focused defaults

## Included Extensions

Snowfox comes with:

* uBlock Origin
* Disable WebRTC
* Don't Track Me Google
* Mullvad Browser Extension
* Trace – Online Tracking Protection
* User-Agent Switcher and Manager

Everything included serves a purpose. No random addons and no unnecessary bloat.

## Build From Source

Snowfox builds the same way as Firefox.

```sh
./mach bootstrap
./mach build
./mach run
```
